# Meal Master - Recipe Finder

## Project Idea and Purpose
Meal Master is a beginner-friendly recipe finder application. Its purpose is to help users easily discover new recipes, view detailed instructions, and manage their weekly shopping lists.

## API Being Used
This project uses the [TheMealDB API](https://www.themealdb.com/api.php) to fetch recipe data.

## Planned Features
1. Search functionality to find recipes by name or ingredient.
2. Recipe details showing ingredients and instructions.
3. A shopping list manager to track required ingredients.
4. Saving favorite recipes for quick access.

## Technologies
- React (via Vite)
- JavaScript (ES6+)
- CSS / HTML
- Git and GitHub
